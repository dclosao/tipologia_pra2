Pràctica 2¶

Presentaci: Hem seleccionat el joc de dades del Titnic donat que compta amb un gran nombre de variables numriques i categriques i el tamany total del joc de dades s bastant bo per a realitzar tasques de processament de dades i d'anlisi exploratori de dades.

Descripció del joc de dades¶

  • Descripci curta: el joc de dades cont informaci sobre el tipus de passatger del Titnic, les caracterstiques del seu viatge i si va sobreviure o no.
  • URL de descrrega: https://data.world/nrippner/titanic-disaster-dataset
  • Mida del joc de dades: 1309 files, 14 columnes
  • Columnes:
    • survival: binari, 0=NO, 1=S. Boole
    • class: tipus de passatger, 1=de primera, 2=de segona, 3=de tercera. Enter
    • name: nom del passatger. Cadena de text
    • sex: sexe del passatger. Cadena de text
    • age: edat en anys. Decimal
    • sibsp: nmero d'esposes o germans. Enter
    • parch: nmero de pares o fills. Enter
    • ticket: nmero del tquet. Enter
    • fare: preu del billet. Decimal
    • cabin: cabina. Cadena de text
    • embarked: port d'embarcament (C = Cherbourg, Q = Queenstown, S = Southampton). Cadena de text
    • boat: bot salva-vides en el cas d'haver sobreviscut. Cadena de text
    • body: nmero del cos en el cas de no haver sobreviscut i que s'hagi trobat el cos. Enter
    • home_dest: destinaci del passatger. Cadena de text

Quina pregunta pretn respondre?

La pregunta principal que tractarem de respondre en aquest projecte s si existeixen relacions entre les caracterstiques del passatgers i el seu ndex de supervivncia.

Objectiu: Per tant, com acabem de comentar l'objectiu principal s conixer si existeixen relacions entre el tipus de passatger i el seu ndex de supervivncia.

# creem una funci que ens compari dues columnes i ens mostri un grfic de barres

def graficar_dues_columnes(df, 
                            col1, 
                            col2, 
                            legloc='upper right',
                            plt_style = 'gggraficar',
                            color_palette="dark", 
                            sorter=None, 
                            stacked=False,
                            kind = 'bar', 
                            percentage = True,
                            custom_title=None, 
                            minimal=True, 
                            figsize=(14,6), width=0.6):   
    
    # creaci de la funci del grfic
    def graficar(table, 
            legloc='upper right',
            plt_style = 'seaborn-ticks',
            color_palette="dark",
            sorter=None, 
            stacked=False,
            kind = 'bar', 
            percentage = True,
            custom_title=None, 
            minimal=True, 
            figsize=(19,10), 
            width=0.7 ):     
        agrupat = table

        if percentage == True:
            agrupat = np.round(agrupat.divide(agrupat['Total'],axis=0)*100,0)
        try:   
            del agrupat['Total']
        except:
            pass

        if sorter:
            agrupat = agrupat[sorter]

        plt.style.use(plt_style)
        sns.set_palette(sns.color_palette(color_palette))
        ax = agrupat.plot(kind=kind,stacked=stacked, figsize=figsize, width=width)
        _ = plt.setp(ax.get_xticklabels(), rotation=0)
        plt.legend(loc=legloc)

        if percentage == True:
          for p in ax.patches:
                ax.annotate('{}%'.format(int(np.round(p.get_height(),decimals=2))),
                                             (p.get_x()+p.get_width()/2.,
                                              p.get_height()), ha='center', va='center',
                                            xytext=(0, 10), textcoords='offset points')
        else:
          for p in ax.patches:
                ax.annotate(np.round(p.get_height(),decimals=2),
                                             (p.get_x()+p.get_width()/2.,
                                              p.get_height()), ha='center', va='center',
                                            xytext=(0, 10), textcoords='offset points')
        if minimal == True:
            ax.get_yaxis().set_ticks([])
            plt.xlabel('')
            sns.despine(top=True, right=True, left=True, bottom=False);
        else:
            pass     
        plt.title(custom_title)
    
    
    
    # creaci del grfic
    agrupat = df.groupby([col2,col1]).size().unstack(col2)
    
    agrupat['Total'] = agrupat.sum(axis=1)
   
    graficar(agrupat, 
            legloc=legloc,
            plt_style = plt_style,
            color_palette=color_palette,
            sorter=sorter, 
            stacked=stacked,
            kind = kind, 
            percentage = percentage,
            custom_title=custom_title, 
            minimal=minimal, 
            figsize=figsize, 
            width=width)
